Poket - IT Services Website

Description: Poket is a responsive IT services website template developed using HTML, SCSS, and Bootstrap. It includes multiple sections like services, portfolio, team, blog, and contact.

Technologies Used:

HTML5
SCSS (CSS)
Bootstrap 3
Font Awesome
Features:

Responsive layout using Bootstrap grid
Navigation bar with popover menu
Banner section with call-to-action buttons
About section with company details
Services section with cards
Portfolio/gallery section
Team members section
Testimonials section
Blog section with modal popup
Contact form UI
Footer with links and social icons
Project Structure:

index.html

css/

bootstrap.min.css
font-awesome.min.css
style.css
js/

bootstrap.min.js
images/

How to Run:

Download or clone the project
Open index.html in a browser
Notes:

SCSS variables are used for styling
Bootstrap components like navbar, modal, and grid are used
Popover is implemented using Bootstrap JS
Purpose: This project is created for learning frontend development and UI design.
